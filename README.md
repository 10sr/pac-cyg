pac-cyg
=======

A command-line installer for Cygwin which cooperates with Cygwin Setup and uses
the same repository. The syntax is similar to `pacman` from Arch Linux.


Synopsis
--------

    pac-cyg <operation> [<option> ...] [<target> ...]
